این یک پروژه Web API ساده و نمونه است که با استفاده از **.NET 10** ساخته شده است. در این پروژه عملیات CRUD (ایجاد، خواندن، به‌روزرسانی و حذف) برای موجودیت محصول (Mahsol) پیاده‌سازی شده است.  

از **Repository جنریک** استفاده شده تا قابلیت استفاده مجدد برای موجودیت‌های دیگر نیز فراهم باشد. همچنین **شناسه (ID)** به صورت جنریک تعریف شده تا بتوان از انواع مختلف شناسه (مانند Guid، int و ...) استفاده کرد.  

در این پروژه، **حذف فیزیکی** محصول انجام نمی‌شود؛ بلکه محصول با تغییر فیلد `Active` به حالت **غیرفعال** (soft delete) درمی‌آید. البته متد حذف (Delete) نیز برای پشتیبانی از این سناریو پیاده‌سازی شده است.  

برای جداسازی لایه‌ها، از **DTOها** استفاده میشد  همچنین **فیلترها و query stringها** برای جستجو و مرتب‌سازی (در صورت نیاز) قابل اضافه کردن هستند.  

این پروژه می‌تواند به عنوان یک الگوی ساده برای شروع پروژه‌های بزرگتر با معماری تمیز و قابلیت توسعه استفاده شود.
